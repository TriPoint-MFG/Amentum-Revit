Amentum Revit

Amentum Revit adds Revit ribbon tools for Excel parameter exchange, CAD round-trip indexing, shared parameter batch workflows, and BIM export — configured for corporate Amentum deployments.

Where to Find It

Restart Revit after installation, then open the Amentum ribbon tab.

Supported Revit Versions

  • Autodesk Revit 2027

Common Workflows

  • Excel Parameter Sync: Import and export Revit element parameters to and from Excel workbooks for bulk editing and reporting.
  • Export 3D DWG: Exports the active 3D view, writes Revit index and 3D metadata sidecars, and posts the companion package to the local Amentum web app when available.
  • CAD Sync: Reads the latest AutoCAD JSON index and moves mapped Revit elements by detected CAD deltas. CAD objects should carry a Revit unique ID in the layer name using RVTUID_..., or matching Revit elements should include the configured CAD handle parameter.
  • Category Shared Params: Generates a Revit shared parameter file grouped by category and normalises common engineering names (e.g. electrical potential → Voltage).
  • Batch Tools: Export family/parameter sheets, apply parameter updates, and merge shared parameter GUID maps.
  • Amentum Web App: Opens the local dashboard at http://127.0.0.1:8765/app when the local service is running.

Installation

Package Scope: This bundle is prepared for Autodesk Revit 2027 deployments.
  1. Copy the Amentum.bundle folder to %APPDATA%\Autodesk\Revit\Addins\<year>\ or to %PROGRAMDATA%\Autodesk\Revit\Addins\<year>\ for a machine-wide deployment.
  2. Restart Revit. The Amentum tab will appear in the ribbon automatically.

Configuration

For local Amentum web workflows, start the Amentum web service and open http://127.0.0.1:8765/app. The Revit add-in posts exports to /api/model-exports and pulls staged edits through /api/revit/sync.

The shared_parameter_map.json file in the Contents folder controls shared parameter name normalisation mappings and can be updated by your BIM manager.
