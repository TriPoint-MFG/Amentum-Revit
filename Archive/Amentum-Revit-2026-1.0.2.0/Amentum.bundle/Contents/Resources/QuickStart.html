Amentum Revit

Amentum Revit adds tools for Excel parameter exchange, Panel View parameter review/editing, shared parameter generation, duplicate project-parameter cleanup, DWG round-trip sync, image export, and scan upload workflows.

Where to Find It

Restart Revit 2026 after installation, then open the Amentum Tools ribbon tab.

Common Workflows

  • Build Parameters: Preview detected parameters in WebView2, check selected rows or check all, create a shared parameter file, and optionally add selected definitions to Project Parameters.
  • Panel View: With selected elements, edit parameters. With no selection, browse project and element/type parameters with thumbnails and model path context.
  • Parameter Jam: Review duplicate project parameter bindings and keep one normalized unique binding set.
  • Export/Import DWG: Export the active 3D view with Revit IDs, then import a CAD JSON index to move matching Revit elements by CAD deltas.
  • Upload Mesh + Pts: Scan once, export OBJ mesh and PLY points, and upload both when configured.

Configuration

shared_parameter_map.json controls shared parameter GUID mappings. Set AMENTUM_UPLOAD_URL for scan uploads and AMENTUM_CAD_TO_REVIT_SCALE when CAD JSON coordinates need unit conversion.

Build Parameters uses Microsoft Edge WebView2. Install the WebView2 Runtime if the preview window does not open.
