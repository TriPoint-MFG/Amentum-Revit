## Short Summary

## Full Description

Amentum Revit is a Revit 2027 add-in designed to streamline model data exchange, CAD coordination, and batch parameter workflows.

The add-in provides tools for importing and exporting element parameters with Excel, generating category-based shared parameter files, synchronizing mapped Revit elements from CAD index changes, and exporting active 3D views to DWG with companion metadata outputs.

Amentum Revit is intended for teams that need practical project data workflows directly inside Revit without leaving the design environment.

## Key Features

- Export selected Revit element parameters to Excel
- Import parameter updates from Excel workbooks
- Export active 3D views to DWG with metadata sidecars
- Apply CAD round-trip sync updates to mapped Revit elements
- Generate category shared parameter files
- Run batch family and parameter update workflows
- Open the local Amentum web dashboard when configured

## Compatibility

- Autodesk Revit 2027
- Windows 64-bit

## Installation

1. Download `Amentum-Revit-2027-1.0.0.0.zip`.
2. Extract the archive.
3. Copy the `Amentum.bundle` folder to one of the following locations:
   - `%APPDATA%\Autodesk\Revit\Addins\2027\`
   - `%PROGRAMDATA%\Autodesk\Revit\Addins\2027\`
4. Restart Revit 2027.
5. Open the **Amentum** ribbon tab.
